# Local PDF to Docs Reader

A small local app that lets you drop in a PDF and read it in a documentation-style page instead of a generic PDF viewer.

## Run

1. Install dependencies:
   - `npm install`
2. Start the app:
   - `npm start`
3. Open:
   - `http://localhost:3000`

## Notes

- Files are processed locally on your machine.
- Works best with text-based PDFs.
- Image-only/scanned PDFs may not extract readable text without OCR.
